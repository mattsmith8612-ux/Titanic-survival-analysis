# Titanic Survival Analysis

Exploratory data analysis of the Titanic dataset using Python, pandas and matplotlib.

## Skills demonstrated
- Python
- pandas
- matplotlib
- Data cleaning
- GroupBy operations
- Missing value analysis
- Data visualisation

## Questions investigated
- How did survival vary by sex?
- How did passenger class affect survival?
- What effect did age have on survival?
- Which passenger groups had the highest survival rates?

## Dataset
This project uses the Titanic dataset from Kaggle's "Machine Learning from Disaster" competition.
