# Titanic Survival Analysis

Exploratory data analysis of the Titanic dataset using Python, pandas and matplotlib to investigate the factors influencing passenger survival.

## Project Overview

This project explores the Titanic dataset using Python, pandas and matplotlib. The analysis investigates passenger demographics, missing data, and the factors that influenced survival aboard the Titanic.
The project was completed as part of my ongoing development in Python and data analysis.

## Skills demonstrated
- Python
- pandas
- matplotlib
- Data cleaning
- GroupBy operations
- Missing value analysis
- Data visualisation

## Questions investigated
- How did survival vary by sex?
- How did passenger class affect survival?
- What effect did age have on survival?
- Which passenger groups had the highest survival rates?

## Dataset
This project uses the Titanic dataset from Kaggle's "Machine Learning from Disaster" competition.

## Files

- `titanic_analysis.py` – Main analysis script.
- `train.csv` – Titanic dataset.
- `passenger_class_frequencies_vertical.png` – Passenger frequencies by class.
- `passenger_class_frequencies_horizontal.png` – Alternative visualisation of passenger frequencies.
- `survival_rate_by_class_and_gender.png` – Survival rates by class and gender.
